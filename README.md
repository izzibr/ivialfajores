# Ivi Alfajores E-commerce 🍪

Bem-vindo ao repositório do e-commerce/catálogo digital da **Ivi Alfajores**. 
Esta é uma aplicação web moderna e responsiva, desenvolvida para apresentar o cardápio de alfajores artesanais e bebidas da marca, permitindo aos clientes explorar os produtos e adicioná-los a um carrinho de compras.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas e tecnologias modernas do ecossistema front-end:

*   **[Next.js](https://nextjs.org/)**: Framework React para renderização otimizada e arquitetura baseada em App Router.
*   **[React 18](https://reactjs.org/)**: Biblioteca para construção da interface de usuário.
*   **[TypeScript](https://www.typescriptlang.org/)**: Superconjunto tipado de JavaScript, garantindo segurança e escalabilidade ao código.
*   **[Styled-Components](https://styled-components.com/)**: Biblioteca de CSS-in-JS para estilização componentizada, dinâmica e isolada.
*   **[Zustand](https://github.com/pmndrs/zustand)**: Gerenciamento de estado global leve e performático (utilizado para o carrinho de compras).
*   **[Framer Motion](https://www.framer.com/motion/)**: Biblioteca para animações fluidas e micro-interações de UI.
*   **[React Icons](https://react-icons.github.io/react-icons/) & FontAwesome**: Para iconografia vetorial consistente.

## ✨ Funcionalidades

*   **Catálogo de Produtos:** Exibição detalhada dos alfajores (Clássico, Amargo, Coco, Triplo, Vegano, etc.) e bebidas.
*   **Carrinho de Compras:** Adição, remoção e controle de quantidade de itens diretamente via interface, utilizando um menu lateral deslizante (CartDrawer).
*   **Design Responsivo:** Interface projetada para funcionar perfeitamente em dispositivos móveis, tablets e desktops, com foco em usabilidade (UX).
*   **Animações Fluidas:** Transições de tela, hover de botões e exibição do carrinho otimizadas para uma experiência de usuário premium.

## 🛠️ Como Executar o Projeto Localmente

Siga as instruções abaixo para rodar a aplicação no seu ambiente de desenvolvimento:

### Pré-requisitos

*   Node.js (versão 16 ou superior)
*   NPM ou Yarn

### Instalação

1. Clone o repositório ou baixe o código-fonte.
2. Acesse a pasta do projeto no terminal:
   ```bash
   cd ivialfajores
   ```
3. Instale as dependências:
   ```bash
   npm install
   ```
   *ou se preferir usar yarn:*
   ```bash
   yarn install
   ```

### Iniciando o Servidor de Desenvolvimento

1. Execute o comando:
   ```bash
   npm run dev
   ```
   *ou*
   ```bash
   yarn dev
   ```
2. Abra o navegador e acesse [http://localhost:3000](http://localhost:3000).
3. Qualquer alteração feita no código será recarregada automaticamente (Hot Reload).

## 📦 Scripts Disponíveis

*   `npm run dev`: Inicia a aplicação em modo de desenvolvimento.
*   `npm run build`: Compila a aplicação para produção (gera a pasta `.next` otimizada).
*   `npm run start`: Inicia o servidor de produção utilizando a build previamente gerada.
*   `npm run lint`: Executa a verificação de qualidade de código (ESLint).
